# Machine Learning House Price Prediction

## 📌 Project Description
This project is a Machine Learning model that predicts house prices based on different features such as area, number of rooms, and other parameters.

The model is built using Python and basic Machine Learning concepts.

## 🚀 Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
